# Clean Approval Blog

Minimal WordPress theme designed for:

- AdSense approval preparation
- Fast loading speed
- Mobile-first responsive layout
- SEO-friendly structure
- Clean readability

## Included Templates

- style.css
- functions.php
- header.php
- footer.php
- index.php
- single.php
- page.php
- archive.php
- sidebar.php
- 404.php

## Installation

1. Download repository ZIP.
2. Upload to `/wp-content/themes/`.
3. Activate from WordPress admin panel.
4. Configure menus and widgets.

## Recommended Plugins

- Rank Math SEO
- LiteSpeed Cache
- WPForms Lite
- Classic Editor

## AdSense Placement

Search for:

```php
<div class="ad-slot">
```

Replace placeholder blocks with your AdSense code.
