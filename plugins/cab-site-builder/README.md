# CAB Site Builder

CAB Site Builder is a lightweight WordPress visual builder framework inspired by GeneratePress, Kadence, Astra, and Blocksy.

## Features

- Header Builder
- Footer Builder
- Popup Builder
- Sticky CTA Bars
- OffCanvas Navigation
- Mega Menu Base
- AI Layout Builder
- Template Library
- Motion Effects
- Dynamic CSS
- Performance Optimization
- Asset Manager
- Reusable CAB Elements
- Conditional Display Rules
- Import / Export System

## Installation

1. Upload `cab-site-builder` to `/wp-content/plugins/`
2. Activate the plugin
3. Open `CAB Builder` in wp-admin

## Recommended Stack

- WordPress 6+
- LiteSpeed Cache
- Redis Object Cache
- Cloudflare CDN

## Builder Workflow

1. Create CAB Elements
2. Assign display rules
3. Import templates
4. Customize layouts
5. Add marketing sections
6. Optimize performance

## AI Builder

Use the AI Builder menu to generate:

- Hero Sections
- CTA Blocks
- Pricing Grids
- Feature Sections
